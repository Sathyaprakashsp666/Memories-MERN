# Memories Creator

 
### Run command git clone https://github.com/Sathyaprakashsp666/Memories-MERN.git

### Run front-end

#### cd client 
#### npm install
#### npm run start || yarn start

### Run backend

#### cd server
#### npm install 
#### npm run start
